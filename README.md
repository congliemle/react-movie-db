## Setup

Prerequisites: 

- [Nodejs](https://nodejs.org/en/)
- [NPM](https://www.npmjs.com)

Run the following command at project's root directory to setup environment

```shell script
$ npm install
```

To run application locally

```shell script
$ npm run start
```

To build application for production usage
```shell script
npm run build
```
